Problem Statement

Why Does This Problem Matter?

🌍

Environmental Degradation

Deforestation, floods, and land degradation are accelerating globally. Monitoring these changes manually is slow and error-prone.

📡

Data Silos

Satellite data from ESA/NASA exists in raw formats (GeoTIFF, COG) that are inaccessible to non-experts without specialized tools.

⏱️

Delayed Response

Traditional methods take days to generate NDVI reports. Real-time environmental risk assessment for disaster prevention is lacking.

🧠

No AI Integration

Existing public tools lack ML-based land-use classification and predictive forecasting for vegetation and climate trends.

Objectives

What We Set Out to Build

  • Real-time NDVI calculation for any GPS coordinate on Earth
  • Interactive map with bounding-box area selection for instant analysis
  • Hourly automated satellite data sync for major Indian cities
  • Deforestation risk detection and flood early-warning system
  • NASA EONET integration for live global disaster event tracking
  • EuroSAT-based ML model for 10-class land-use classification
  • NDVI trend forecasting using scikit-learn regression models
  • Secure multi-user access with OAuth 2.0 / Spring Security
  • Professional PDF report export for environmental analysis
  • Premium React dashboard with real-time charts and alerts

Core Features

What SDAVS Can Do

🌿

NDVI Analysis

Real-time vegetation health index for any coordinate. Formula: (NIR−Red)/(NIR+Red)

Interactive Map

Leaflet map with click-to-analyze and bounding-box area selection.

🤖

Machine Learning

ML & AI Intelligence Module

🏷️ EuroSAT Image Classifier

Classifies satellite images into 10 land-use categories using color-statistics similarity against 500 EuroSAT RGB training samples. Includes anti-screenshot validation checks.

Forest River AnnualCrop Industrial SeaLake Residential

📈 NDVI Trend Forecasting

scikit-learn regression models predict future NDVI values and vegetation trends. Trained on historical Sentinel-2 time-series data.

⚙️ COG Processing Pipeline

Python + rasterio processes Cloud Optimized GeoTIFFs (COG) from the STAC API. Extracts spectral bands (NIR, Red, Green, Blue), computes NDVI, and returns geospatial metadata.

Data Flow

How Data Flows Through SDAVS

🔄 Automated Pipeline (Hourly)

  • Scheduler triggers for all major Indian cities
  • Spring Boot queries Element84 STAC for latest Sentinel-2 tiles
  • Python COG service downloads and processes GeoTIFF bands
  • NDVI values + metadata persisted to Supabase PostgreSQL
  • Risk flags calculated and stored for dashboard display

🖱️ On-Demand User Flow

  • User clicks location or draws bounding box on map
  • Frontend sends coordinates to Spring Boot REST API
  • Backend queries STAC API for that region's latest imagery
  • Python ML service classifies image and computes NDVI
  • Results returned to UI with charts and risk assessment
